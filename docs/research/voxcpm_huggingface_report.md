# VoxCPM2 HuggingFace 模型研究报告

## 1. 模型概览

| 属性 | 值 |
|------|------|
| **模型地址** | https://huggingface.co/openbmb/VoxCPM2 |
| **版本** | VoxCPM2 (当前推荐版本) |
| **参数量** | 2B |
| **架构** | Tokenizer-Free Diffusion Autoregressive |
| **许可证** | Apache-2.0 |
| **训练数据** | 200万+小时多语言语音数据 |

## 2. 技术架构

### 2.1 模型架构详情

| 组件 | 描述 |
|------|------|
| **整体架构** | LocEnc → TSLM → RALM → LocDiT |
| **语言模型基础** | MiniCPM-4 |
| **音频编解码** | AudioVAE V2（非对称编解码） |
| **输入采样率** | 16kHz |
| **输出采样率** | 48kHz |
| **数据类型** | bfloat16 |
| **LM Token Rate** | 6.25 Hz |
| **最大序列长度** | 8192 tokens |
| **VRAM 需求** | ~8 GB |

### 2.2 核心创新点

1. **Tokenizer-Free 设计**
   - 绕过离散token化，直接在连续空间建模
   - 避免token化带来的信息损失

2. **Diffusion Autoregressive 架构**
   - 结合扩散模型的生成质量和自回归模型的序列建模能力
   - 通过 LocDiT（局部扩散Transformer）实现高质量音频生成

3. **AudioVAE V2**
   - 非对称编解码设计
   - 内置超分辨率功能，无需外部升采样器
   - 16kHz输入 → 48kHz输出

4. **隐式语义-声学解耦**
   - 通过分层语言建模和FSQ约束实现
   - 增强表现力和生成稳定性

## 3. 功能特性

### 3.1 核心功能

| 功能 | 描述 |
|------|------|
| **30语言多语言支持** | 无需语言标签，直接输入任何支持的语言文本 |
| **声音设计** | 仅通过自然语言描述生成全新音色，无需参考音频 |
| **可控克隆** | 克隆音色后可通过风格指导控制情感、节奏和表达 |
| **极致克隆** | 提供参考音频及其转写文本，实现最高保真度克隆 |
| **48kHz 工作室品质输出** | 内置超分辨率，无需外部升采样器 |
| **上下文感知合成** | 自动从文本内容推断合适的韵律和表达方式 |
| **实时流式生成** | RTX 4090 上 RTF 低至 0.13（Nano-VLLM） |

### 3.3 API 使用示例

**基础文本转语音:**
```python
from voxcpm import VoxCPM
import soundfile as sf

model = VoxCPM.from_pretrained("openbmb/VoxCPM2", load_denoiser=False)

wav = model.generate(
    text="VoxCPM2 brings multilingual support, creative voice design, and controllable voice cloning.",
    cfg_value=2.0,
    inference_timesteps=10,
)
sf.write("output.wav", wav, model.tts_model.sample_rate)
```

**声音设计:**
```python
wav = model.generate(
    text="(A young woman, gentle and sweet voice)Hello, welcome to VoxCPM2!",
    cfg_value=2.0,
    inference_timesteps=10,
)
```

**可控克隆:**
```python
# 基础克隆
wav = model.generate(
    text="This is a cloned voice generated by VoxCPM2.",
    reference_wav_path="speaker.wav",
)

# 带风格控制的克隆
wav = model.generate(
    text="(slightly faster, cheerful tone)This is a cloned voice with style control.",
    reference_wav_path="speaker.wav",
    cfg_value=2.0,
    inference_timesteps=10,
)
```

**极致克隆:**
```python
wav = model.generate(
    text="This is an ultimate cloning demonstration using VoxCPM2.",
    prompt_wav_path="speaker_reference.wav",
    prompt_text="The transcript of the reference audio.",
    reference_wav_path="speaker_reference.wav",
)
```

**流式生成:**
```python
import numpy as np

chunks = []
for chunk in model.generate_streaming(text="Streaming is easy with VoxCPM!"):
    chunks.append(chunk)
wav = np.concatenate(chunks)
```

```bash
    --config_path conf/voxcpm_v2/voxcpm_finetune_all.yaml
```

## 8. 限制与风险

### 8.1 技术限制

| 限制 | 描述 |
|------|------|
| **声音设计和风格控制** | 结果可能在多次运行间有所变化，建议生成1-3次以获得期望输出 |
| **语言性能差异** | 性能因语言而异，取决于训练数据可用性 |
| **长文本稳定性** | 非常长或高度表达性的输入可能出现不稳定 |
| **直接控制有限** | 当前版本对特定语音属性（如情感或说话风格）的直接控制有限 |

### 8.2 使用限制

| 限制 | 描述 |
|------|------|
| **严禁用途** | 冒充、欺诈或传播虚假信息 |
| **AI生成标记** | 强烈建议公开分享的内容明确标记为AI生成 |
| **生产环境** | 不建议在未经严格测试和安全评估的情况下用于生产或商业应用 |
