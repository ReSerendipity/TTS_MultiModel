## 二、发现的问题（按严重程度排序）

### 🔴 严重问题 (Critical)

#### BUG-01: 国际化功能完全失效
- **描述**: 设置 `?lang=en` 参数后，页面所有 UI 文本仍然显示中文。导航按钮、标题、表单标签、按钮文字等均未翻译。
- **复现步骤**:
  1. 访问 `http://127.0.0.1:7869/?lang=en`
  2. 观察页面所有文本
- **预期**: 所有 UI 文本应显示英文（如 "Voice Design"、"Voice Clone" 等）
- **实际**: 所有文本仍为中文（"声音设计"、"语音克隆" 等）
- **截图**: `test_12_i18n_bug_en_not_working.png`
- **影响**: 英文用户完全无法使用英文界面

#### BUG-02: 设置 API 返回非 JSON 内容导致前端解析失败
- **描述**: `/api/system/settings` 接口返回了非 JSON 内容（可能是 "Internal Server Error" HTML 页面），导致前端 `JSON.parse()` 抛出 `SyntaxError`。
- **控制台错误**: `Failed to fetch settings: SyntaxError: Unexpected token 'I', "Internal S"... is not valid JSON`
- **影响**: 设置页面（引擎信息、GPU VRAM、缓存统计）的数据无法正确加载和显示

### 🟡 中等问题 (Medium)

#### BUG-03: 音频播放器 src 设置为 null
- **描述**: 生成请求完成后，前端尝试设置音频播放器的 `src` 属性为 `null`，导致 `TypeError`。
- **控制台错误**: `TypeError: Cannot set properties of null (setting 'src')`
- **影响**: 用户无法听到生成的音频

#### BUG-04: 极致克隆高级参数全部为只读状态
- **描述**: 极致克隆页面的所有高级参数（CFG 值滑块、推理步数滑块、使用随机种子复选框等）均标记为 `readonly`，用户无法修改这些参数。
- **复现步骤**:
  1. 切换到极致克隆 Tab
  2. 尝试点击"使用随机种子"复选框或拖动 CFG 滑块
- **预期**: 用户应能调整这些参数
- **实际**: 所有参数为 readonly，无法交互
- **影响**: 用户无法自定义生成参数，降低了功能可用性

#### BUG-05: Vite 开发服务器客户端脚本引用残留
- **描述**: 页面中引用了 `@vite/client` 脚本（Vite HMR 热更新客户端），但应用运行在 Uvicorn 生产模式下，导致该请求返回 404/ERR_ABORTED。
- **网络请求**: `GET http://127.0.0.1:7869/@vite/client type=Script failed=net::ERR_ABORTED`
- **影响**: 每次页面加载产生一个无效的网络请求，虽不影响功能但不规范

### 🟢 轻微问题 (Minor)

#### BUG-06: Health Check 轮询过于频繁
- **描述**: `/api/system/health` 端点被高频轮询，在约 100 个网络请求中，超过 60 个是 health check。估算轮询间隔约为 1-2 秒。
- **影响**: 增加不必要的网络和服务器负载，建议调整为 5-10 秒间隔

#### BUG-07: SSE 连接在页面刷新时全部中断
- **描述**: 页面刷新或导航时，所有 SSE 连接（progress、status、cancel、engine_switch）都返回 `ERR_ABORTED`。虽然这是浏览器正常行为，但前端应实现自动重连机制。
- **影响**: 如果用户在生成过程中刷新页面，SSE 进度推送将丢失

#### BUG-08: 生成按钮缺少前端空文本验证
- **描述**: 在声音设计页面，即使文本框可能为空，点击生成按钮仍会发送请求并进入加载状态（按钮变为"生成中..."并禁用）。没有前端 `required` 验证阻止提交。
- **影响**: 产生不必要的网络请求，依赖后端验证返回错误

#### BUG-09: 输出结果区域可能重复渲染
- **描述**: 在生成请求后，页面出现了两个"输出结果" heading（ref=e29 和 ref=e42），说明 HTMX 可能将新内容追加到了已有内容后面，而非替换。
- **影响**: 页面布局可能出现重复元素

## 五、改进建议

1. **[紧急] 修复国际化功能**: 检查 `i18n.py` 中的 `get_lang()` 函数和模板中的 `lang` 变量传递逻辑，确保 `?lang=en` 能正确切换界面语言
2. **[紧急] 修复设置 API**: 检查 `/api/system/settings` 端点，确保始终返回有效 JSON
3. **[高优先级] 修复音频播放器 null src**: 在设置 `audio.src` 前添加 null 检查
4. **[高优先级] 检查极致克隆参数 readonly**: 确认是否为设计意图，如果不是则移除 readonly 属性
5. **[中优先级] 移除 Vite 客户端引用**: 在生产构建中移除 `@vite/client` 脚本引用
6. **[中优先级] 优化 health check 轮询频率**: 建议从 1-2 秒调整为 5-10 秒
7. **[低优先级] 添加前端表单验证**: 在生成按钮点击时检查文本框是否为空
8. **[低优先级] SSE 自动重连**: 实现 EventSource 的自动重连机制
