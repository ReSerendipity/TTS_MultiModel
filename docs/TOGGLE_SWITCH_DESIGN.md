# 开关样式设计报告

## 当前设计（已实施）

所有开关已采用 **双色调圆形滑动开关** 设计：

### 开启状态（Checked）
- **左侧（白色）**：开关轨道的左半部分为白色
- **右侧（绿色）**：开关轨道的右半部分为 #10B981（绿色）
- **圆点**：白色圆点（#fff）带阴影（box-shadow: 0 1px 4px rgba(0,0,0,0.4)），滑动到右侧
- **视觉效果**：左白右绿，白色圆点在右侧，清晰表明"开启"状态

### 关闭状态（Unchecked）
- **左右两侧（灰色）**：开关轨道整体为 #6b7280（灰色）
- **圆点**：白色圆点带阴影，滑动到左侧
- **视觉效果**：整体灰色，白色圆点在左侧，清晰表明"关闭"状态

## 实现方式

使用 CSS `linear-gradient` 实现左白右绿的分割效果：
- 开启：`linear-gradient(to right, #fff 0%, #fff 48%, #10B981 52%, #10B981 100%)`
- 关闭：`linear-gradient(to right, #6b7280 0%, #6b7280 48%, #6b7280 52%, #6b7280 100%)`

圆点通过 `transform: translateX(18px)` 实现平滑滑动动画。

## 修改的文件

1. `settings.html` - 所有开关的 HTML 结构和 CSS 样式
2. `settings.html` - `initToggles()` JavaScript 函数，处理开关状态切换
