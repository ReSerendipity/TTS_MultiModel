# TTS_MultiModel 项目可调参数分析报告

## 概述

经过对项目代码的全面审查，共发现约 **219 个参数**，其中 **66 个** 已通过 UI 暴露给用户，其余 **153 个** 为内部使用。本报告按类别整理所有可调参数，标注其默认值、范围、以及当前是否暴露给用户。

### 2.3 默认生成参数

这些参数控制 TTS 引擎的基础生成行为，属于全局配置，通常在系统启动时加载。

| 参数名称 | 默认值 | 范围 | 当前 UI | 说明 |
|---------|--------|------|--------|------|
| **默认采样率** (default_sample_rate) | 24000 Hz | 建议 ≥16000 | ❌ | 定义输出音频的采样率，单位为赫兹。24000Hz 是大多数 TTS 模型的标准采样率，可提供良好的音质。较低值（如 16000Hz）可减少文件大小但可能影响音质，较高值会增加计算开销但音质提升不明显。 |
| **默认语速** (default_speed) | 1.0 | (0, 3.0] | ❌ | 控制语音输出的默认速度倍率。1.0 表示正常语速，0.5 表示半速播放（更慢），2.0 表示双倍速播放（更快）。范围上限为 3.0，超过此值可能导致语音失真或计算错误。此参数作为后端默认值，用户可在各 Tab 的后处理区域通过"变速因子"滑块进行调整。 |
| **默认随机种子** (default_seed) | 42 | 任意整数 | ❌ | 用于控制语音生成的随机性。相同种子值在相同输入条件下会产生相同的输出结果，便于复现特定语音效果。默认值 42 是一个常用的种子值。设为 0 表示完全随机，每次生成结果不同。此参数作为系统默认值，用户可在 IndexTTS2 Tab 中手动指定种子值或启用随机种子。 |
| **剧本段间静音** (script_studio_silence_secs) | 0.4s | (0, 2.0] | ❌ | 控制剧本工坊（Script Workshop）中多角色对话段落之间的静音间隔时长。较长的间隔（如 0.8s）可使对话更自然、更具戏剧感，较短的间隔（如 0.2s）可使对话更紧凑。超过 2.0s 可能导致对话不连贯，低于 0.1s 可能导致段落切换过于突兀。此参数仅影响剧本工坊功能。 |

## 九、参数可暴露性建议

### 9.1 建议暴露给用户的参数

| 参数名称 | 当前状态 | 建议 UI | 位置 |
|---------|---------|--------|------|
| 每段最大字符数 | ❌ | 数字输入框（只读，固定值200） | 设置页-高级参数 |
| 最大生成长度 | ❌ | 数字输入框（只读，固定值3000） | 设置页-高级参数 |
| 重试时长比率阈值 | ❌ | 数字输入框 | 设置页-高级参数 |
| 空闲超时 | ❌ | 数字输入框 | 设置页-高级参数 |
| 目标响度 (LUFS) | ❌ | 滑块 (-30 ~ 0) | 各 Tab 后处理区域 |
| 引擎总字符上限 | ❌ | 只读显示 | 各 Tab 输入框下方 |

### 9.2 不建议暴露的参数

以下参数属于系统底层调优参数，不建议暴露给用户：

- 缓存容量/条目数（应自动管理）
- GPU 内存检查间隔（应自动管理）
- 内存泄漏检测阈值（应自动管理）
- 时间估计器参数（应自动学习）
- 音频后处理底层参数（削波阈值、压缩比等）
- 服务器/网络参数（应由部署环境管理）
- LoRA 训练参数（已通过独立 API 管理）
