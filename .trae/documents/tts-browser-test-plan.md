# TTS MultiModel Voice Studio — 全面浏览器测试计划

## 摘要

对运行在 `http://127.0.0.1:7869/` 的 TTS MultiModel Voice Studio Web 应用进行全面的浏览器自动化测试。该应用是基于 FastAPI + Jinja2 + HTMX 架构的 AI 语音合成平台，支持声音设计、声音克隆、极致克隆、剧本工坊、音色库管理、LoRA 训练等功能。

## 当前状态分析

- **技术栈**：FastAPI 后端 + Jinja2 模板 + HTMX 前端 + SSE 实时推送
- **UI 架构**：侧边栏导航 + Tab 页面内容区，9 个功能 Tab
- **国际化**：支持中文 (zh-CN) 和英文 (en)，通过 Cookie + URL 参数持久化
- **响应式**：4 个断点 (1200px / 900px / 768px / 480px)
- **测试工具**：agent-browser (MCP integrated_browser)

## 假设与决策

1. **应用已启动**：假设 `http://127.0.0.1:7869/` 已正常运行，模型已加载
2. **测试浏览器**：使用 agent-browser (Chromium) 进行自动化测试
3. **测试数据**：使用简单的中文文本进行生成测试，不依赖特定音色文件
4. **不测试实际音频生成质量**：重点测试 UI 交互和功能流程，不评估 TTS 输出质量
5. **截图记录**：每个关键步骤截图保存，便于问题追溯

## 验证方式

- 每个测试步骤通过浏览器快照 (snapshot) 验证页面状态
- 关键步骤截图保存到工作目录
- 发现的问题记录在测试报告中，包含截图和复现步骤
- 最终输出完整的测试报告，汇总所有发现的问题
